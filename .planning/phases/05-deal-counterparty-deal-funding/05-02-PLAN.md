**File 2: `backend/services/counterparties.py`**

Create with `from __future__ import annotations` at top.

Key implementation details:

1. **Aliased joins** (follow DealService pattern exactly):
```python
TierRef = aliased(RefData)
InvestorTypeRef = aliased(RefData)
CpartyCompany = aliased(Company)
```

2. **`_base_stmt(self, deal_id: UUID)`** method:
```python
stmt = (
    select(
        DealCounterparty,
        CpartyCompany.name.label("company_name"),
        TierRef.label.label("tier_label"),
        InvestorTypeRef.label.label("investor_type_label"),
    )
    .outerjoin(CpartyCompany, CpartyCompany.id == DealCounterparty.company_id)
    .outerjoin(TierRef, TierRef.id == DealCounterparty.tier_id)
    .outerjoin(InvestorTypeRef, InvestorTypeRef.id == DealCounterparty.investor_type_id)
    .where(
        DealCounterparty.deal_id == deal_id,
        DealCounterparty.org_id == self.current_user.org_id,
    )
)
```

3. **`list_for_deal(self, deal_id: UUID, page: int = 1, size: int = 50)`** — default page size 50 per D-16. Use `clamp_pagination`, `count_rows`, `page_count` from `_crm`. Order by `DealCounterparty.position.asc().nullslast(), DealCounterparty.created_at.asc()`.

4. **`_counterparty_response(row)`** static method — maps row to DealCounterpartyResponse. Access `row[0]` for the ORM object, `row.company_name`, `row.tier_label`, `row.investor_type_label` for joined labels. Convert Decimal fields to float with None guard (same as DealService pattern).

5. **`create(self, deal_id: UUID, data: DealCounterpartyCreate)`** — verify deal exists and belongs to org first (`_get_deal_or_404`). Create DealCounterparty with `org_id=self.current_user.org_id`, `deal_id=deal_id`, and all Create fields. Flush, commit, re-query with `_base_stmt` to return joined response.

6. **`update(self, deal_id: UUID, counterparty_id: UUID, data: DealCounterpartyUpdate)`** — load counterparty by id+deal_id+org_id. Apply all non-None fields via `model_dump(exclude_unset=True)` loop + setattr. Commit and re-query.

7. **`delete(self, deal_id: UUID, counterparty_id: UUID)`** — load, verify org_id, delete, commit.

8. **`_get_deal_or_404(self, deal_id: UUID)`** — select Deal where id=deal_id and org_id=current_user.org_id. Raise 404 if not found.
